# Telegram VPN Bot

![VPN BOT](welcome.png)

Телеграмм-бот для продажи VPN-подписок с ручным подтверждением платежей, управлением серверами, промокодами и автоматической выдачей конфигурационных файлов.

# Скачать и запустить одной командой
```bash
wget -O - https://raw.githubusercontent.com/thetemirbolatov-official/Telegram-Bot-VPN/main/install.sh | sudo bash
```

## 🌟 Особенности

### 📋 Основные функции
- **Продажа VPN-подписок**: 1 месяц, 3 месяца, 6 месяцев
- **Ручное подтверждение платежей**: Админ проверяет и подтверждает платежи
- **Промокоды**: Бесплатные пробные периоды
- **Мультисерверность**: Поддержка нескольких серверов (Германия, Нидерланды)
- **Автоматическая выдача конфигов**: WireGuard конфигурации
- **QR-коды**: Быстрая настройка через сканирование

### 👤 Пользовательские функции
- Покупка и продление подписок
- Активация промокодов
- Просмотр активных ключей
- Инструкции по установке для Android и iOS
- Автоматические уведомления об окончании подписки
- Техническая поддержка

### 🔧 Административные функции
- Статистика продаж и пользователей
- Управление платежными методами
- Загрузка и управление конфигурационными файлами
- Массовая загрузка конфигов
- Рассылка сообщений пользователям
- Управление промокодами
- Экспорт данных в Excel
- Резервное копирование и восстановление
- Синхронизация конфигов

## ⚙️ Настройка бота

### 1. Настройка конфигурации
Откройте файл `vpn.py` и настройте следующие параметры:

```python
# Конфигурация бота
TOKEN = 'your_bot_token_here'  # Токен от @BotFather
ADMIN_ID = 123456789  # Ваш ID в Telegram
CHANNELS = [
    "@your_channel_1",  # Обязательные каналы для подписки
    "@your_channel_2",
]
```

### 2. Настройка серверов
По умолчанию бот поддерживает два сервера. Вы можете изменить их в разделе `servers_db`:
```python
servers_db = {
    'server1': {
        'name': '🇩🇪 Germany',
        'location': 'Germany, Frankfurt',
        'load': 'Low',
        'protocol': 'WireGuard',
        'ip': 'vpn-de1.example.com',
        'available_configs': [],
        'used_configs': {}
    },
    'server2': {
        'name': '🇳🇱 Netherlands',
        'location': 'Netherlands, Amsterdam',
        'load': 'Low',
        'protocol': 'WireGuard',
        'ip': 'vpn-nl1.example.com',
        'available_configs': [],
        'used_configs': {}
    }
}
```

### 3. Настройка тарифов
Тарифные планы настраиваются в переменной:
```python
SUBSCRIPTION_PLANS = {
    '1 месяц': {'price': '199₽', 'days': 30},
    '3 месяца': {'price': '499₽', 'days': 90},
    '6 месяцев': {'price': '999₽', 'days': 180}
}
```

## 🐛 Поиск и устранение неисправностей

### Распространенные проблемы:

1. **Бот не запускается**
   - Проверьте токен бота
   - Убедитесь, что все зависимости установлены
   - Проверьте права на запись в директорию

2. **Не отправляются конфиги**
   - Проверьте наличие .conf файлов в директории
   - Убедитесь, что файлы имеют правильные названия
   - Используйте функцию "🔄 Синхронизировать конфиги"

3. **Платежи не подтверждаются**
   - Проверьте наличие пользователя в базе
   - Убедитесь, что есть свободные конфиги на сервере

4. **Ошибки с изображениями**
   - Убедитесь, что все необходимые изображения присутствуют
   - Проверьте права на чтение файлов

### Логи
Все ошибки записываются в `vpn_bot.log`. Проверяйте этот файл для диагностики проблем.
